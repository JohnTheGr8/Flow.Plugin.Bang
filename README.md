Wox.Plugin.Bang
==================

A plugin for the [Wox Launcher](https://github.com/Wox-launcher/Wox)

### About

Take advantage of [DuckDuckGo's bangs](https://duckduckgo.com/bang) to search on thousands of websites directly.
